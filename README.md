# REquestionaire

Static real estate website questionnaire for routing visitors to the right next step.

## Files

- `index.html` - page structure and content
- `styles.css` - site styling
- `script.js` - questionnaire flow, routing logic, and saved state
- `real-estate-questionnaire.md` - planning and content reference
- `transfer.md` - original transfer guide
- `version-notes/` - numbered change log for project updates

## Run Locally

Open `index.html` in a browser.

## Current Flow

1. Choose visitor segment
2. Choose property type
3. Choose main objective
4. Choose timeline
5. Show recommended route and saved intake data
